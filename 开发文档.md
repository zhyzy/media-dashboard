# 遇见约到家 · 新媒体流量监测平台（可视化数据大屏）开发文档

---

## 一、项目概述

### 1.1 项目背景
为统一监控新媒体部门在各平台的流量数据，构建一个集**数据录入 + 实时同步 + 可视化展示**于一体的数据大屏系统。

### 1.2 项目目标
- 汇总多平台流量数据
- 支持后台手动录入数据
- 前端大屏实时更新（无需刷新）
- 提供可视化分析与排行榜

---

## 二、系统架构

### 2.1 架构说明
前端（数据大屏） <——> 后端服务（API + WebSocket） <——> 数据库
(Vue3 + ECharts) (Node.js / Java) (MySQL)
### 2.2 技术选型

#### 前端
- Vue3 + Vite
- ECharts（图表库）
- Pinia（状态管理）
- WebSocket（实时通信）

#### 后端
- Node.js（Express / NestJS）或 Java（Spring Boot）
- JWT 登录认证
- Socket.IO（推荐）

#### 数据库
- MySQL
- Redis（可选，用于缓存）
